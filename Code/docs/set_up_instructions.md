|----------------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| Code Style/Formatter | **Settings > Code Style > Kotlin > Scheme**: </br> [Code Style Settings](https://github.com/devrath/RunTracer/blob/main/Code/docs/images/kotlin-code-style.png)|
